# Changelog

## v0.1.0

Initial public launch package.

- MCP server for FIX/trading-ops diagnostics and recovery workflows.
- Mission Control dashboard with scenario catalog, case brief, operator rail, workbook, trace, FIX wire, terminal, and copilot.
- 13 trading-operations scenarios across session, order, market-data, reference-data, regulatory, and algo incidents.
- 22 MCP tools covering diagnostics, session recovery, order actions, venue/reference operations, algo controls, stress injection, scoring, and trace evidence.
- Docker Compose stack for local demo: console, API server, PostgreSQL, Redis, and MCP stdio service.
- Remotion-based demo walkthrough assets and captured MCP evidence for BATS startup recovery.
